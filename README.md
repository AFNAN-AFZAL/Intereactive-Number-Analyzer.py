# Intereactive-Number-Analyzer.py
Building/Generating/Developing an Interactive Number Analyzer using Conditional Statements and Logical Operators (Basic Level)

Simple README: How it Works
•	while True: This is a "forever loop." It keeps the program running so you can check more than one number.
•	break: This is the emergency exit. When you type "exit," the loop stops and moves to the summary at the bottom.
•	numbers_list.append(num): This is like dropping a coin into a jar. Every time you enter a number, it gets saved in the list.
•	len(numbers_list): This simply counts how many items are in your list.
